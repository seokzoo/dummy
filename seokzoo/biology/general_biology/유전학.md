유전자 : DNA 상에 RNA로 전사되는 정보를 담고 있는 부분
DNA가 이루는 구조들
	히스톤 단백질로 DNA를 감아 놓고 있음(뉴클레오솜). 히스톤 단백질은 그래서 염기성 아미노산 중 리신과 아르기닌이 많음 (8량체 구조) DNA를 그냥 히스톤에 감아 놓으면 불안정하니까 히스톤 H1으로 고무줄같이 잡아줌 (히스톤 H1은 뉴클레오솜에 포함안됨) 뉴클레오솜 단위체로 쭉 연결된 것을 염색사(간기)라고 부름. 핵 속에서 염기성 염색약(메틸렌 블루, 아세트산카민, 헤마톡실린)으로 염색되는 부분을 염색질(DNA, RNA, 단백질 등이 포함됨)이라고 함
		진정염색질(Euchromatin) : 덜 응축된 곳으로 유전자 전자가 활방함
		이질염색질(Heterochromatin) : 응축되서 유전자 전사 잘 안되는 곳
	분열기가 되면 DNA를 복제하고 염색분체 형태롤 응축해놓고 코헤신 단백질로 붙여놓음. 코헤신으로 붙은 부분을 중심절이라고 부름. 중심절에 동원체라는 방추사가 붙는 단백질 복합체가 있음. 염색분체 두 개가 붙은 걸 염색체라고 함(그냥 한 벌짜리 염색사도 염색체라고 부름). 염색분체의 각 끝에는 텔로미어가 있음.
	사람은 23 쌍의 염색체(총 46개)를 가지고 있음
		상 염색체 : 22쌍으로 남녀 공통으로 가지며 크기, 모양 동일
		성 염색체 : 여자는 XX, 남자는 XY를 가지고 Y염색체가 조금 더 작음
	상동 염색체 : 모양과 크기가 같은 한 쌍의 염색체
	대립 유전자 : 삼동 염색체 상에서 같은 위치에 같은 형질을 결정하는 유전자들
		우성, 열성에 따라 어떤 것이 발현될지 결정됨
	핵상 : 하나의 세포 속에 들어있는 상동 염색체의 벌 수와 염색체 수
		n = 23, 2n = 46
	핵형 : 세포에 들어있는 염색체의 수, 크기, 모양 등으로 핵형을 분석해서 개체나 돌연변이를 파악할 수 있음
세포주기
	체세포 분열 : 각 단계로 넘어가기 전에 검문지점(checkpoint)을 지나야함
		간기 : DNA가 염색사 형태로 존재하고 세포 주기의 대부분을 차지
			G1기 : 세포가 성장하는 시기, 에너지를 많이 얻기 위해 미토콘드리아, 엽록체 늘어남. 세포 마다 G1기의 길이는 다름
			S기 : DNA를 복제하고 코헤신으로 붙여 놓음. 중심체 복제 시작
			G2기 : 세포가 성장하는 시기로 자유 튜불린 합성 많이 함
				MPF(Mitosis promoting factor) : G2->M checkpoint에 관여하는 단백질
					사이클린 B(주기적으로 양이 증감하는데 M기 전에 최고로 많아짐) + Cdk1 결합으로 구성 (사이클린 B의 양이 충분히 많아지면 결합(조절 단백질 조절)해서 kinase작용함)
						라민 인산화 -> 핵막 소실
						콘덴신 인산화 -> 염색체 응축
						미오신 인산화 -> 수축환의 조기 생성 방지
					cf) 세포 분열 억제 인자들 : 외부 신호, DNA 손상, 밀도 의존적 억제, 부착 의존성
		분열기(M기) : 염색사가 염색체 형태로 응축하고 분열이 일어남
			핵분열
				전기 : 콘덴신이 염색체를 응축시킴. 응축되면서 인 사라짐. 중심체가 분리되어 세포 끝으로 이동하면서 성상체가 됨
				전중기 : 라민 인산화되면서 핵막 사라짐. 방추사가 동원체에 부착
				중기 : 동원체-방추사가 염색체를 적도판에 배열함 (핵형 분석하기 좋은 시기)
				후기 : 코헤신이 분해되면서 염색분체가 세포 끝으로 끌려감
					방추사가 염색분체에 연결되는 동원체 방추사가 있고 반대편 성상체에서 나오는 방추사끼리 연결되는 극성 방추사가 있음. 극성 방추사는 서로 연결된 상태로 밀어내서 성상체가 밀려나게 됨. 세포막 쪽에 연결되서 앵커 역할을 하는 성상체 방추사도 있음
				말기
			세포질 분열
				동물세포 : 미세섬유가 수축환을 만들어 세포를 서서히 조임
				식물세포 : 골지체 유래 소낭을 극모에서 나온 방추사를 따라 적도판에 배열하고 격막형성체를 만드는데 격막형성체가 세포벽으로 바뀜
		휴지기(G0기) : 분열을 멈춘 세포가 해당하고 분열 유도 신호가 없으면 여기에 머무름. 성인의 대다수 세포가 여기에 속함. G0일 때만 Cdk 없음
	감수 분열
		M기만 다름
			M1 : 사분 염색체 형성
				전기에 접합복합체를 이용하여 상동 염색체를 결합해 사분 염색체(2가 염색체)를 만듬(붙을 때 대립유전자끼리 전부 맞 닿게 붙임) 남자도 XY가 모양이 다른데 사분 염색체 형성함(서로 붙을 수 있는 자리가 있음) 사분 염색체 형성하면 서로 재조합 일어남
				후기에 슈고신이 코헤신이 분해되지 않게 막아줘서 염색체 상태로 끌려옴(상동염색체 분리)
			M2 : 염색분체로 분리
				후기에 코헤신 분리되고 염색분체로 분리됨
		교차 현상 : 사분 염색체 형성될 때 상동 염색체 쌍 사이에 DNA 일부가 교환됨 (가로, 대각선 상으로도 DNA 교환이 일어날 수 있음) 교차되는 지점은 교환점(키아즈마)이라고 부름
체세포 분열 : 핵상 변화 없음
	e.g.) 모세포(2n) -> 딸세포(2n)
감수 분열(생식 세포 분열) : 핵상이 절반됨
	e.g.) 모세포(2n) -> 딸세포(n)

악성 종양(암)
	양성종양(혹) : 국소 종양으로 천천히 자라며 고유의 기능을 유지하고 전이가 거의 일어나지 않음
	암 : 빠르게 자라며 탈분화되어 고유 기능을 잃고 주변으로 전이됨
		분열 조절 x, 밀도 의존적 억제 x, 새로운 혈관 형성, 주변으로 전이, 부착 비의존적, 스스로 성장 인자 방출
		원인 : 돌연변이 물질, 유전자 파괴, 종양 바이러스
		치료 : 수술, 방사선 치료, 항암제
		세포사멸
			세포괴사 : 물리적 자극에 의해 세포가 죽는 현상으로 염증 반응이 일어남
			세포자살 : 신호에 의해 세포 자살 유전자가 활성화되면 세포가 죽는 현상으로 염색질이 응축되고 세포막 파괴없이 세포가 기포화되고 DNase가 히스톤에 감겨있는 DNA를 규칙적으로 자름 -> 염증 반응없이 남은 물질들을 재활용함
				원인 : 산소 고갈, 미세소관 파괴, DNA 파손, 텔로미어 마모, 생존 신호 결여
멘델의 유전 법칙
	멘델의 가설
		형질을 결정하는 한 쌍의 유전 인자가 있다
		우성 유전 인자가 발현되고 열성 유성 인자는 미발현된다
	기본 용어들
		대립 유전자 : 상동 염색체의 같은 위치(좌위)에 존재하며 한 가지 형질을 결정하는 유전쌍
		유전형 : 형질을 나타내는 유전자의 조합 e.g.) Aa, Bb
		표현형 : 겉으로 나타나는 형질
		야생형 : 집단에서 가장 보편적인 유전자형
		돌연변이형 : 유전자 발현이 바뀐 변이체
		동형접합자(순종) : 상동염색체의 대립 유전자 쌍이 동일한 경우
		이형접합자(잡종) : 상동염색체의 대립 유전자 쌍이 다른 경우
		단성교배 : 한 가지 형질을 결정하는 한 쌍의 대립 유전자를 대상으로 하는 교배
		양성교배 : 두 가지 형질을 결정하는 두 쌍의 대립 유전자를 대상으로 하는 교배
		자가교배 : 같은 유전자형끼리의 교배
		자가수분 : 같은 꽃 꽃가루가 암술에 수분되는 것
		타가수분 : 다른 꽃 꽃가루가 암술에 수분되는 것
		치사 : 발현되면 죽는다는 뜻
	멘델의 법칙
		우열의 법칙 : 우성의 인자만 표현형으로 나타남
		분리의 법칙 : 감수 분열 시 대립 인자는 서로 분리됨. 따라서 이형접합자 끼리 교배 시 표현형 비율이 1:3으로 나타남
		독립의 법칙 : 다른 형질에 관여하는 유전자는 유전될 때 서로 독립적으로 분리됨
		cf) 검정교배 : 개체를 열성 동형 접합자랑 교배시키는 것; 우성 표현형이랑 열성 동형접합자랑 교배시키면 우성 표현형이 순종인지 잡종인지 알 수 있음 -> 우성 동형 접합자였으면 우성 표현형만 나옴
		예외들
			불완전 우성(중간 유전) : 상동 염색체 상의 두 유전형 모두가 형질에 영향을 주는 경우
			공동 우성 : 이형 접합자의 두 대립 유전자의 유전형이 표현형으로 모두 나타나는 경우
			복대립 유전 : 대립 유전자가 3개 이상 존재하는 경우 e.g.) ABO식 혈액형(적혈구 막의 당 구조)
				A형 : N-아세틸갈락토오스, B형 : 갈락토오스
			다면 발현 : 한 개의 유전자가 서로 관련 없어 보이는 여러 형질에 영향을 주는 현상
			유전자 상호작용 : 독립인 서로 다른 좌위의 유전자들이 같은 형질을 결정하는 일련의 대사 과정에 관여하는 경우 형질 분리비가 9:3:3:1이 안 나올 수도 있음
			다인자 유전 : 유전자 상호작용이 없는(같은 대사과정에 관여x) 다른 좌위의 유전자들이 하나의 형질에 영향을 주는 현상
		성 결정 염색체
			사람, 초파리 : XX, XY
				Y 염색체에 있는 SRY 유전자가 발현되면 원시생식선이 정소로 발달함
				XO : 터너 증후군
				XXY, XXXY, XXXXY : 클라인펠터 증후군
				반성 유전 : 성 염색체 상에 유전자가 있는 것
				성염색체 보정 : 여성의 경우 X염색체가 하나 더 많기 때문에 비슷한 수준의 유전자 발현을 맞추기 위해서 별도의 보정이 필요함.
					X 염색체 불활성화(바소체) : 암컷이 포배기가 됐을 때 배아 줄기 세포(ESC)에서 관찰됨
						cf) 체세포 : 생물체를 이루는 세포들, 배세포 : 감수 분열을 통해 생식 세포를 형성하는 세포들
						각 세포에서 X 염색체 쌍 중 하나가 랜덤으로 선택되고 Xist라는 유전자에서 RNA를 발현시킴 해당 RNA가 자신을 발현시킨 염색체를 감싸서 응축시킴 (응축된 염색체를 바소체라고 부름) -> 개체가 자라나서 배세포를 만들면 그 때 응축이 풀림
			조류 : ZW, ZZ
			벌, 개미 : 반수체형(핵상이 2n이면 암컷), 미수정란이 수컷이 됨
		연관
			독립 : 서로 다른 두 유전자 좌위가 서로 다른 염색체 상에 있는 경우
			연관 : 서로 다른 두 유전자 좌위가 같은 염색체의 팔에 위치한 경우
				상인 : 연관된 두 유전자 모두 우성 또는 열성인 경우
				상반 : 우성과 열성 유전자가 서로 연관된 경우
				완전 연관 : 두 유전자 거리가 매우 짧아서 유전자 사이에서 교차가 일어나지 않는 경우
				불완전 연관 : 두 유전자 거리가 멀어서 사분염색체 형성할 때 교차가 일어날 수 있음
					A와 B의 교차율 (p < 0.5) = A와 B가 교차된 것 수 / 전체 집합 수 (상인으로 가정하고 풀면 쉬움) 
						p가 0.5면 독립임`
						교차가 된 유전자 형이 나올 확률 p/2 안된 유전자 형이 나올 확률 (1-p)/2
						교차율과 유전자 지도 작성
							교차율 0.01을 1 지도 단위(1 cM; centiMorgan)로 봄
							실제 물리적 거리랑 교차율을 비교하면 완전 비례하지는 않음 <- 여러번 교차될 수도 있고 반복되는 서열은 교차가 잘 일어나고 단백질이 붙어있으면 교차가 잘 안일어남
	게놈 각인
		포유류 염색체 중 20-30개의 유전자에서 나타남
		난소 또는 정소에서 생식 세포를 만들 때 모계 또는 부계에서 유래한 특정 대립 유전자가 항상 메틸화된 상태로 전해져서(수컷이 A를 메틸화하면 암컷은 A를 메틸화하지 않는 식) 자손에서 해당 유전자가 발현이 되지 않는 현상(체세포 분열 할 때도 메틸화가 유지되는구나). 자식이 자라서 배세포를 만들면 탈메틸화를 시키고 성별에 따라 다시 메틸화함
	모계유전(세포질 유전) : 난자 세포질에 있는 미토콘드리아, 엽록체가 형질을 결정하는 경우
	유전 질환
		알비노증(백색증) : 멜라닌 색소를 만드는 유전자 이상으로 피부, 털이 하얗다
		페닐케톤뇨증 : 페닐알라닌 대사에 문제가 생겨서 체내 페닐알라닌 축적
		낭포성 섬유증 : 상피 세포의 염소 이온 분비에 이상이 생겨 기도 등에 점액이 축적됨(염소 이온에 의한 삼투압으로 점액이 촉촉해지는데 점액이 마르니까 배출이 안됨)
		겸상 적혈구 빈혈증 : 아미노산 하나가 잘못돼서 이상하게 생긴 헤모글로빈이 만들어지고 적혈구 안에서 뭉쳐져서 적혈구 모양이 ㄱ모양으로 변형됨 -> 산소를 잘 못 옮김, 지라(비장)에서 해당 적혈구는 쉽게 파괴됨 -> 만약에 해당 유전자를 잡종으로 가지고 있으면 스트레스(말라리아 감염, 산소 부족)를 받는 환경에서만 낫 모양으로 찌그러짐
	질병이 우성, 열성 질환인지 파악하는 방법
		정상 부모 밑에서 환자가 태어나면 열성질환
		환자 부모 밑에서 정상 태어나면 우성질환
		상, 성 염색체 연관 질환인지는 직접 대입해보면 됨
유전 물질의 연구
	DNA 연구의 역사
		그리피스의 실험 - 형질 전환의 발견
			생쥐에 살아있는 S형 균주를 넣으면 죽음
			생쥐에 살아있는 R형 균주 넣으면 삶
			생쥐에 가열한 S형 균주 넣으면 삶
			생쥐에 가열한 S형이랑 살아있는 R형 넣으면 죽음 -> 생쥐 관찰 결과 살아있는 S형 발견됨
		에이버리, 맥러드, 매카시의 실험
			가열한 S형, 살아있는 R형의 경우 S형에 DNase를 처리했을 때만 생쥐가 살았음(RNase, 단백질분해효소는 죽음)
		허시, 체이스의 실험 - DNA가 유전물질임을 확립
			단백질에 35S로 표지한 바이러스와 DNA에 32P로 표지한 바이러스를 만듦
			각각을 세균에 감염을 시키고 믹서기로 갈아 서로 떨어트림
			각각을 원심분리 하고 어디에 방사선 표지가 되어 있는지 확인
				단백질 : 위쪽(바이러스)에 방사선 표지가 되어 있음
				DNA : 아래쪽(세균)에 방사선 표지가 되어 있음 -> DNA가 유전 물질일 가능성 높음
	DNA 구조의 분석
		샤가프의 법칙
			생물에서 DNA를 추출해서 염기 조성을 확인하여 퓨린과 피리미딘의 총량이 같다는 사실을 알아냄
		프랭클린, 윌킨슨
			X선 결정학 실험으로 DNA의 원자 배열을 관찰하여 반복적인 구조를 확인함
		왓슨, 크릭
			X선 결정 사진을 통해서 DNA의 분자 구조 모델을 구축함
			두 가닥이 서로 역평행하게 놓여 있고 인산 백본을 구성해 전체적으로 음전하를 띔 (염기성 염색약(메틸렌블루, 아세트산카민, 헤마톡실린)으로 염색 가능)
			판으로 쌓인 각 염기는 소수성을 띄어서 소수성 결합으로 이어져 있음
			오른 나선 구조로 되어있음
	DNA의 복제 방법
		메셀슨, 스탈 : DNA가 반보존적 복제를 하는구나
			원본 DNA를 15N으로 표지하고 새로 합성되는 DNA를 14N으로 표지함
			각 세포주기 별 세포를 밀도차 원심분리를 해서 14N, (14+15)N, 15N DNA를 분리함
		원핵생물의 복제 : 세타 폼 복제라고 부르고 복제시 복제 버블을 형성함
			도구들
				헬리케이스 : ddDNA를 외가닥으로 분리시킴
				SSBP(single strand binding protein) : 풀린 외가닥에 붙어서 떨어진 상태를 유지시킴
				프라이메이스 : 복제 개시할 때 RNA 프라이머를 합성(3'-OH를 제공)
				DNA Polymerase : 주형 가닥과 상보적으로 DNA를 합성(주형 가닥 기준 3'->5', 신생 DNA 기준으로 5'->3' 방향으로 합성 진행, 3'-OH가 있어야 합성을 시작할 수 있음(RNA polymerase는 없어도 가능)
					DNA Polymerase III : 긴 가닥 DNA를 합성하고 3'->5' 핵산말단가수분해 효소가 있어서 교정 기능이 있음
					DNA Polymerase I : 짧은 가닥 DNA를 합성하고 3'->5' 핵산말단가수분해 효소도 있고 앞에 있는 장애물을 부술 수 있는 5'->3' 핵산말단가수분해 효소도 있음
						핵산말단가수분해 효소(Exonuclease) : DNA의 끝부분을 자를 수 있는 효소
						핵산내부가수분해 효소(Endonuclease) : DNA의 중간 부분을 자를 수 있는 효소
					(DNA Polymerase I, III는 원핵에만 있음)
				리가아제(ligase) : 3'-OH랑 5'-인산 부분(nick)을 연결해줌
				자이레이스(Gyrase; Topoisomerase의 일종) : ATP를 사용해서 DNA의 과도한 꼬임을 풀어줌
			진행 : 원형DNA의 복제는 Origin(대장균은 Ori C, A-T rich)라는 지점에서 시작됨 -> 헬리케이스가 ATP를 사용해서 DNA를 벌리고감 -> SSBP가 붙어서 DNA 열린 상태 유지 -> 프라이메이스가 RNA 프라미어를 합성 -> DNA Polymerase III가 DNA를 합성함 -> 각 ssDNA에 lagging strand(오카자키 절편 형성)랑 leading strand가 붙게 됨 -> 오카자키 절편의 3'-OH에 DNA Polymerase I이 붙고 5'->3' 핵산말단가수분해 효소가 작용해서 프라이머를 부숨과 동시에 3'->5' 방향으로 DNA를 합성함 -> 생성된 nick(DNA Polymerase가 합성한 DNA의 3' 부분과 그 앞에 5' 부분 사이) 을 ligase가 붙임 -> Ter site라는 곳까지 Polymerase가 복제를 하면 종결됨
				replication fork : dsDNA가 분리되는 지점을 말하고 복제 버블 형성시 2개가 생김
				dNTP에 달린 피로인산(PPi)이 떨어지면서 ssDNA에 연결됨
		진핵생물의 복제 : 선형 DNA
			복제원점이 여러개가 있음 (복제원점 별로 개시가 되는 시점이 다를 수 있음)
			나머지 과정들은 원핵생물의 DNA 복제 과정이랑 비슷한데 새로 합성된 가닥의 5' 말단 부분은 덜 합성됨
				텔로머레이스 (RNA(참조 역할, 리보자임 아님) + 단백질 -> 역전사) : RNA를 주형으로 DNA 가닥을 합성 (주형 가닥의 3' 말단 쪽에 붙어서 상보적인 DNA를 합성함) -> 자기가 만든 서열에 끝 부분에 다시 붙어서 DNA를 합성할 수 있음 -> 텔로미어 생성
		PCR(Polymerase chain reaction) : 소량의 DNA로 특정 DNA 절편을 증폭시키는 방법
			cf) in-vitro(시험관), in-vivo(생체 내)
			DNA 복제 과정을 시험관에서 구현 (아래 과정 반복)
				1. 가열(약 94도)해서 dsDNA를 분리함 (변성, denaturation)
				2. DNA 프라이머를 넣어줌 
				3. 냉각하면 DNA프라이머가 ssDNA에 붙음 (hybridization)
				4. 가열(72도)하면 Taq DNA Polymerase(고온에서 변성안됨)가 dNTPs를 이용해서 DNA를 합성
		아가로오스 겔 전기영동 (DNA판 SDS-PAGE) : 폴리아크릴아마이드 겔 처럼 그물 구조고 이것도 짧으면 더 멀리감
		DNA 지문법 : 염색체 내 특정 위치들에 일부 염기 서열들이 반복적으로 나타나는데 반복 정도가 사람마다 다름 (유전되기 때문에 친자 확인에도 사용) -> 반복되는 부분 바깥에 사람마다 똑같은 서열이 나타나는 경우가 있는데 그 부분에 맞는 프라이머를 짜서 PCR을 돌림
유전자의 발현
	기본 용어 정리
		붉은 빵곰팡이 : 핵상이 n이라서 돌연변이가 일어나면 바로 표현형으로 확인할 수 있음
		세균이나 균류 베지 만드는 법 : 샬레에 아가로오스 겔에 영양 물질을 넣고 굳힘
		완전 배지 : 생물이 살아가는데 필요한 모든 영양분을 제공
		최소 배지 : 생물이 살 수 있는 최소한의 영양분만 제공(포도당 + 무기물)
		야생형 : 자연 상태의 생물
		영양 요구주 : 성장에 필요한 특정 물질을 합성하지 못하는 돌연변이(최소 배지에 키우면 죽을 수 있음)
		전구 물질 : 합성 물질의 재료
	역사적 배경
		비틀, 테이텀의 실험
			X선을 붉은 빵곰팡이에 쪼여서 아르기닌 영양 요구주를 만들어냄 -> 아르기닌의 전구물질을 투입하면 돌연변이에 따라 생존하는 것도 있고 아닌 것도 있음 -> 아르기닌은 포도당을 3단계에 걸쳐서 아르기닌으로 전환하는데 각 과정 마다 관여하는 효소가 있음
			(X선은 단백질을 변성시키지 않음)
				결론 : X선이 각 효소의 합성을 지시하는 유전자를 망가뜨렸을 것이다 -> 1 유전자 1 효소설 제창 -> 추후 1유전자 1폴리펩티드설로 발전 -> 선택적 스플라이싱이 밝혀져서 1 유전자 여러 폴리펩티드설로 발전
		중심 원리 - 크릭
			유전 정보의 흐름을 정리한 이론 (복제 -> 전사 -> 번역)
		최근의 이론
			RNA가 DNA로 역전사할 수도 있음
			RNA 바이러스나 바이로이드는 RNA를 유전 물질로 가지고 복제를 할 수 있음
			프리온이라는 스스로 복제할 수 있는 단백질도 있음
	전사(Transcription)
		유전자에는 암호화 부위와 프로모터라는 부분이 있음 -> RNA Polymerase가 주형 가닥의 프로모터 부분에 붙음 (주형 가닥의 상보적인 가닥인 코드 카닥은 새로 합성되는 RNA와 서열이 같음 그래서 mRNA를 코돈 가닥이라고 부름)
		유전자 : DNA에 유전 정보가 담긴 부분으로 조절 부위(전사를 조절)와 암호화 부위(RNA로 전사)가 있음. 암호화 부위의 첫 번째 부분을 +1이라고 하고 그 바로 전 부분을 -1이라고 해서 인덱스를 매길 수가 있음. 유전자 서열을 언급할 때는 코드 가닥으로 언급하는 것이 규칙임 -> 따라서 TATA 박스나 그런 것들도 전부 코드 가닥 기준으로 말하기 때문에 실제 붙는 곳은 TATA박스에 상보적인 주형 가닥임
		원핵생물
			프로모터 프리나우박스(-10, -35 지점)에 RNA Polymerase(알파_2베타베타프라임(핵심기능) + 시그마 인자; 완전효소라고 부름)가 붙음. 시그마 인자는 RNA Polymerase가 프리나우박스에 붙을 수 있도록 도와주는 역할을 하기 때문에 DNA에 붙고 나면 다시 떨어질 수 있음. 스스로 dsDNA를 벌려서 전사를 개시할 수 있음(합성되는 RNA 기준 5'->3')
			전사 종결
				Rho-dependent termination : 전사된 RNA에 Rho 단백질이 붙는 자리가 있어서 거기 붙은 다음 RNA Polymerase를 쫒아감 -> GCGCGC같은게 이어진 부분이 나오면 루프를 만들 수가 있음 -> 루프가 생기면 NusA라는 단백질이 붙어서 루프와 RNA Pol을 잡음(RNA Pol이 더 이상 못감) -> RNA Pol이 Rho에 닿으면 해체됨
				Rho-independent termination : RNA에 GCGCGC에 의한 루프가 형성되고 바로 UUUUUU가 나오면 NusA가 고리와 RNA Pol을 붙잡고 결국 RNA가 DNA에 뜯겨 나오면서 종결됨
		진행생물
			RNA 합성효소 종류
				RNA Pol I : 인에서 전구 rRNA 합성
				RNA Pol II : 핵질에서 mRNA, miRNA 합성
					많은 전자인자들(전사를 조절하는 단백질)과 전사 복합체를 형성함
					과정 : -40, -20(-20에 TATA 박스 있음) 부분에 RNA Polymerase II가 붙음
				RNA Pol III : 핵질에서 5s rRNA, tRNA 합성
			전사 과정
				RNA Pol II에 여러 전사인자들이 붙은 전사 복합체가 프로모터에 붙음 -> 뉴클레오티드 하나를 NTP 상태로 DNA에 붙임 -> 그 후는 NTP에서 나오는 에너지를 이용해서 NMP를 계속 붙임 (NTP-NMP-...-NMP 이런 식으로 연결된 거임)
				전사 종결
					mRNA 3' 말단에 AAUAAA이 합성되면 핵산내부가수분해 효소가 붙어서 약 20nt 떨어진 지점에서 RNA를 자르면 RNA Pol II가 mRNA에서 분리됨
mRNA의 구조
	원핵의 mRNA
		폴리시스트로닉(Polycistronic) : 한 개의 mRNA에 여러 개의 유전 정보가 담겨있음 -> 같은 대사에 관여하는 효소들이 한 번에 전사를 할 수 있음
			샤인-달가노 서열 : 리보솜이 붙는 부분
			5', 3'-UTR : 양 끝에 번역 안되는 부분
			스페이서 : 각 유전자 정보 사이에 있는 간격
	진핵의 mRNA : 엑손(유전자 정보)과 인트론(번역 안되는 부분으로 보통 GU, AG로 시작하고 끝남)이 있음
		1차 전사체에서 3가지 가공 과정을 거침 -> 가공된 RNA를 성숙된 RNA(mature RNA)라고 부름
			G-Cap 형성 효소가 5' 말단에 G-Cap 을 씌움
				mRNA가 분해되지 않도록 안정화
				G-Cap이 있어야 세포질로 나갈 수 있음
				40S 리보솜이 결합하는 자리 역할
			3' 말단에 Poly A tail을 연장함
				mRNA가 분해되지 않게 안정화
				세포질로 나가는 것을 도와줌
				꼬리가 길면 번역량이 더 많아짐 -> 진핵생물은 번역을 할 때 어떤 효소가 G-Cap과 Poly A Tail을 붙잡아서 고리 구조를 만드는데 고리가 있으면 반복적으로 번역하기 더 쉽기 때문에 전사량이 늘어남
			RNA Splicing (불필요한 인트론 제거) - 리보자임에 의해
				스플라이소좀 복합체(snRNP + snRNA(리보자임으로 실직적인 기능을 함)) : 인트론 부분에 붙어서 인트론을 올가미 모양으로 때냄
				자가 스플라이싱 : 일부 인트론은 스플라이소좀 도움없이 알아서 떨어져나감
					2군 인트론 : 스플라이소좀 처럼 올라미 모양으로 떨어짐
					1군 인트론 : GMP에 도움을 받아서 선형 형태로 떨어짐

번역(Translation)
	DNA의 코드 가닥과 같은 mRNA의 코돈의 정보에 따라 리보솜이 단백질을 합성해냄
	tRNA에는 안티코돈 팔이 달려있어서 코돈에 상보적으로 결합할 수 있음
		안티코돈 : 코돈에 대응하는 tRNA의 유전 암호
		세포에 각 아미노산에 대응하는 20종류의 이미노아실-tRNA 신세테이스가 있음(ATP 사용) -> tRNA에 아미노산을 붙여주는 역할을 해줌(tRNA의 3' 말단 CCA 부분에 아미노산이 붙음)
	트리플렛 코드와 아미노산 : 세 개의 염기가 하나의 아미노산을 지정함
		트리플렛 코드는 거의 모든 생물에서 잘 보존되어 있음(개시 : AUG(Met), 종결 : UGA, UAG, UAA)
	cf) 니렌버그의 실험
		Poly U, A, C mRNA를 각각 합성해서 단백질을 만들어냄 -> 각각에 대응해서 합성된 단백질을 확인 -> 이런식으로 트리플렛 코드를 완성함
	워블 가설 : 안티코돈의 3' 말단 쪽의 2nt 부분은 아주 특이적으로 붙음 그런데 안티코돈의 5' 말단 쪽의 1nt 부분은 훨씬 덜 특이적으로 붙음 (축퇴성이 있다고 함 - 3번째는 대충 와도 된다)
	원핵생물의 번역 (E, P, A 자리가 있음) : 전사와 번역이 동시에 일어남
		샤인-달가노 서열에 30S 리보솜이 붙음 -> AUG를 찾음 -> fMet이 붙은 tRNA가 와서 그걸 30S 리보솜 P자리에 붙임 -> 50S 리보솜이 씌워짐 -> A 자리에 새로운 tRNA 들어옴 -> 23S rRNA가 아미노산과 아미노산 사이의 펩티드 결합을 만들어줌 (단백질은 N->C 말단 방향으로 합성되고 리보솜이 3nt씩 전진함) -> 종결코돈에 도달하면 tRNA 대신 RF(Releasing factor)라는 인자가 와서 리보솜 복합체를 해체시킴
		(리보솜이 작동하는 데 사용되는 에너지는 모두 GTP임)
	진핵생물의 번역
		단백질이 G-Cap과 Poly A Tail를 붙잡아 loop를 만듦 -> Met이 결합한 60S 리보솜이 G-Cap에 붙음 -> 40S 리보솜이 그 위를 덮어줌 -> 스캔을 시작해서 코작서열(거기 안에 AUG가 포함되어 있음)을 찾음
	원핵, 진핵 둘 다 하나의 mRNA에 여러 리보솜이 붙어서 폴리 리보솜을 구성하여 동시에 여러 번역을 할 수도 있음
폴리펩티드의 이동
	자유리보솜 : 단백질을 세포질에서 만드는 경우
		세포질에서 쓰이는 경우 : 폴리펩티드에 특별한 신호서열 없으면 이렇게 되고 알아서 접히거나 셰퍼론 단백질의 도움을 받아서 접힘
		핵 안에서 쓰이는 경우 : 폴리펩티드 어딘가에 NLS(Nuclear Localization Signal) 서열이 있음 -> 임포틴 단백질이 해당 단백질을 핵 속으로 수송함 (NLS는 보통 폴리펩티드 가운데 쯤에 있어서 안자름)
		엽록체나 미토콘드리아 내부에서 사용되는 경우 : 목적지에 따라 기질 신호서열이 N말단(대부분 N말단에 신호서열이 오면 나중에 다 잘라냄)에 있는데 그걸 인식해서 셰퍼론 단백질이 붙어서 단백질이 안 접히게 만들어줌 -> 선형 상태로 미토콘드리아 막 구멍으로 집어 넣음 -> 미토콘드리아 기질에서 신호 서열을 자름 (엽록체의 경우 틸라코이드 내부까지 이동시키는 경우에 N 말단에 틸라코이드-스트로마-N말단 이런식으로 신호 서열이 추가로 더 있음)
	결합리보솜
		분비 단백질 만들기
			소포체까지 : 자유리보솜처럼 번역을 하다가 N말단 신호서열이 나오면 거기게 SRP가 붙는데 그러면 번역 멈춤 -> RER의 SRP 수용체에 결합하면 트랜스로콘 구멍이 열리고 번역이 진행되면서 구멍 안으로 단백질이 합성됨 -> 신호서열 절단 -> PDI가 이황화결합 시켜줌 -> 돌리콜에 당을 붙여놨다가 아스파라긴에 당을 옮겨 붙여줌(N 연결 당화)
			소포체에서 쓰려면 그냥 그 상태로 쓰면 되고 아니면 골지체로 이동
			골지체부터 : 일부 당 제거, 세린과 트레오닌에 당 결합(O 연결 당화) -> 붙은 당에 따라 단백질의 최종 목적지가 정해짐
		막 단백질 만들기
			N말단 신호서열이 있을 수도 없을 수도 있음 근데 막관통 이동 종결 신호서열이 있음(소수성 서열) -> 해당 서열이 있으면 소포체로 번역이 되면서 들어가다 소수성을 띄는 막관통 이동 종결 신호서열이 소포체 막에 박힘 -> 박힌 상태로 소포체 안에 나온 부분은 가공이 이루어짐(N연결 당화, 신호서열 제거, 이황화결합) -> 소포체에서 소낭 형태로 그래로 떨어져 나와서 미세소관 타고 골지체로 가서 융합되고 거기서 또 가공(당화, 당제거)일어나고 다시 소낭으로 떨어져나와서 세포막에 융합됨
			(소낭 안쪽 부분에서 당 가공이 일어나는데 그래서 세포막 바깥쪽에서 당들(당지질, 프로테오글리칸)이 관찰되는 거임)
바이러스
	박테리오파지 : 세균을 숙주로 삼는 바이러스
		용균 파지 : 용균주기만 가능 e.g.) T4 파지
		용원 파지 : 용균주기 + 용원주기 둘 다 가능 e.g.) 람다 파지
			용균주기 : 세균에 부착해서 자기 DNA를 집어 넣음(단백질이랑 유전물질이 분리되므로 이 때는 바이러스라고 안봄, 즉 바이러스 수는 0이 되는 거임) -> 세균DNA를 파괴하고 자신의 DNA를 복제함 -> 코트 단백질을 만들고 자기 자신을 조립함 -> 세균벽 분해 효소를 만들고 세균벽을 파괴하고 탈출
			용원주기 : 파지의 DNA가 세균의 DNA에 끼어들어가서 몇 세대 정도 같이 복제되며 살아감 (프로파지, 프로바이러스 라고 부름) -> 숙주가 복제하면 바이러스도 복제되는 효과가 있음 -> 양분이 풍부한 곳이나 세균이 UV에 쪼여지면 다시 용균주기를 시작함
			cf) 플라크 : 세균들이 자라는 플레이트에 바이러스가 퍼져서 세균이 없는 부분 <-> 콜로니
	cf) RNA 바이러스는 외가닥 유전물질을 가지고 있구나 -> 더 불안정 (U, C 전환도 쉬움)
	동물 바이러스
		cf) 코드 가닥, 코돈 가닥과 똑같은 서열을 가진 mRNA를 (+) RNA라고 부르고 그것에 상보적인 mRNA를 (-) RNA라고 부름
		인플루엔자 바이러스 : 독감의 원인 <-> 감기는 리노바이러스, 코로나바이러스에 의해 걸림
			(-) RNA 8개가 유전 물질임 그걸 뉴클레오켑시드가 감싸고 있음
			NA(뉴라민 분해효소)
			HA(혈구응집소) : 세포막에 있는 시알산이라는 당이랑 결합함 -> 세포 안쪽으로 엔도시토시스됨 -> 바이러스 지질막이 엔도좀 막이랑 융합하면서 바이러스 RNA랑 일부 효소가 세포질 안으로 배출됨 -> 바이러스 핵산이 세포 핵 안으로 들어감 -> RNA replicase(생물은 안가지고 있는 효소)로 (+) RNA를 만들어냄 -> 한번 더 해서 (-) RNA를 만들어서 복제함 -> 전에 나온 (+) RNA로 번역해서 코트 단백질 및 효소를 만들어냄 -> 바이러스 막에 있는 NA나 HA는 소포체, 골지체 통해서 만들어냄
		HIV 바이러스 : 레트로 바이러스
			AIDS를 일으킴
			동일한 유전 정보를 지닌 (+) RNA 2개를 가지고 있음
			역전사 효소, 삽입효소, 단백질 분해효소를 가지고 다님
			RNA가 세포 안으로 들어오면 역전사 시켜서 cDNA(RNA를 역전사시켜서 만든 DNA)를 만듦 -> DNA 2개가 합쳐져서 dsDNA를 만듦 -> 핵 안으로 들어가고 삽입효소가 프로바이러스 형태로 만들고 숙주 RNA Polymerase II가 전사해서 번역 후 필요한 단백질 만듦 -> 단백질 분해효소가 만들어진 단백질 쪼개고 각각이 조립돼서 바이러스 다시 완성
전이인자(Transposable element)
	여기 저기 옮겨다니는 유전자
	DNA 트랜스포존(Transposon)
		전위효소(Transposase)가 들어있어서 숙주 염색체 안에서 무작위로 움직일 수 있음 <- 양 끝의 IR(Inverted Repeat)을 인식해서, 잘라낸 다음 끼워 넣을 수도 있고 복사해서 다른데 끼워 넣을 수도 있음 -> 유전자 사이에 껴들어가서 돌연변이를 일으킬 수도 있음
		맥클린톡의 트랜스포존 발견
			옥수수의 C유전자는 붉은 색소 유전자고 그 사이에 고장난 전이인자인 Ds가 껴있으면 흰 옥수수됨. 근데 다른 곳에 정상 전사인자 Ac가 있으면 걔가 Ds를 다른데로 옮겨주면 C유전자가 다시 정상적으로 발현되기 때문에 얼룩덜룩한 옥수수가 되는 거임
	레트로 트랜스포존(Retrotransposon)
		양 끝에 LTR(Long Terminal Repeat)이 있음 -> 역전사 효소를 가지고 있음 -> 그게 전사, 변역되면 걔가 mRNA를 역전사해서 cDNA를 만듦 -> 그게 다시 무작위로 DNA에 끼어 들어감
		이게 레트로 바이러스의 조상으로 추측하고 있음
박테리아
	성장곡선
		지연기(Lag phase) : 환경에 적응하는 시기(필요한 유전자를 발현) e.g.) 먹이가 포도당에서 락토오스로 바뀌는 경우
		지수성장기(Exponential phase) : 세균들의 특성이 비슷해서 연구하기 좋은 시기 (균일한 결과를 얻을 수 있음)
		정지기(Stationary phase) : 세포 분열과 세포 사멸 속도가 평형을 이루는 시기 (전체 세포수 변화가 없는 시기)
		사멸기(Death phase) : 세포수가 감소하는 시기
	박테리아의 유전 정보 교환 (gene transfer) - 수평 유전자 전달
		유전자 도입(Transgenesis)
			형질전환(Transformation) : 다른 세균이 죽으면서 방출된 DNA조작이 다른 세균 속으로 들어가면 RecA라는 효소가 재조합해서 DNA조각이 세균 DNA 안으로 들어감 (그냥 세균이 다른 DNA(플라스미드 포함)를 삼키면 형질전환임)
			형질도입(Transduction) : 바이러스가 숙주 DNA를 부수고 자기를 복제하는 과정에서 코트 단백질안에 숙주 DNA 조각이 들어감 -> 그게 다른 세균에 부착해서 DNA가 안으로 들어가면 똑같이 RecA가 재조합시켜줌
			접합(Conjugation) : 두 세균이 성선모를 이용해 직접 DNA를 전달함
				cf) 플라스미드 : 숙주 DNA와 별개로 있는 작은 원형 DNA (일종의 기생 DNA)
					R 플라스미드 : 항생제에 저항성을 줌
					F 플라스미드 : 회전환(Rolling-circle) 복제를 할 수 있음
						바깥 ssDNA를 자른 다음 5' 말단 쪽을 밖으로 내보내 풀면서 3'말단 쪽에서는 안쪽 ssDNA를 참조해서 복제를 함 -> 한 바퀴가 다 복제가 되면 자른 다음 5' 쪽에 프라이머를 붙여서 dsDNA를 만들고 양쪽을 붙여서 원형 플라스미드를 만듦
				F 플라스미드가 가지고 있는 유전자가 발현되면 선모가 생겨서 F 플라스미드가 없는 세균한테 뻗어짐 -> 세균끼리 연결되면 F 플라스미드가 회전환 복제를 해서 복사본을 건내줌 -> 선모 끊어짐
				Hfr 균주 : F 플라스미드에도 트랜스포존이 끼어들어갈 수 있는데 그러면 플라스미드랑 세균의 DNA 사이에 재조합이 일어날 수 있음 -> 이렇게 재조합된 세균을 Hfr 균주라고 부름 -> 이게 다른 세균이랑 접합이 일어날 수 있는데 그러면 일부 플라스미드 DNA에 일부 세균 DNA(너무 길이가 길어서 전부 넘어갈 수는 없음)가 딸려갈 수 있는데 RecA에 의해 딸려간 세균 DNA랑 다른 세균의 DNA 간에 재조합이 일어날 수 있음 -> 전부 넘어간 것은 아니기 때문에 F 플라스미드의 유전자는 못 받음
	박테리아 유전자 발현의 조절
		오페론 : 원핵세포 유전자의 특징으로 프로모터, 작동자 서열(operator), 여러 개의 구조 유전자(mRNA로 전사되는 유전자 부위) 등으로 구성된 전사 단위
			 Lac 오페론 : (i : 따로 떨어져있는 유전자로 작동자 서열에 붙는 억제자 단백질을 만듦), CAP, Promoter, Operator, Lac Z(베타-갈락토시데이즈 - 젖당을 포도당이랑 갈락토오스로 분해하는 효소), Lac Y(락토오스 투과효소 - 세포 안으로 락토오스를 들이는 효소), Lac A
				 글루코스있고 락토오스 없을 때 : 작동자 서열에 붙은 억제자 단백질이 순간 떨어질 때 전사가 일어날 수 있음
				 글루코스있고 락토오스 있을 때 : 젖당이 분해되면서 알로락토오스(유도물질)라는 물질이 되는데 그게 억제자 단백질에 결합해서 작동자 서열에 못 붙게 함. 프로모터가 마음대로 붙을 수 있긴 한데 그렇게 전사가 많이 일어나지는 않음
				 글루코스없고 락토오스 있을 때 : Glc이 없으면 cAMP가 많아지는데 cAMP가 CAP(어딘가 다른데서 발현됨)에 결합해서 CAP서열에 붙음 -> RNA Polymerase의 전사를 촉진시켜줌
				 젖당은 분해하는데 유전자를 3개나 발현해야되기 때문에 에너지를 더 많이 쓰게 되고 그 만큼 성장 속도가 느려진다.
			 트립토판 오페론
				 (trpR : 따로 떨어져있는 효소로 트립토판(보조억제자)과 결합해서 작동자 서열에 결합해서 전사를 방해함)프로모터, 작동자 서열, 트립토판 합성하는 효소 유전자들
					 혹시 억제자가 잠깐 떨어졌을 때 RNA Polymerase가 전사를 해버리면 엄청 에너지 낭비임 그래서 감쇠자 조절을 함
						 감쇠자 조절 : 구조 유전자 부분 앞인 Leader 부분부터 전사가 됨 -> 리더 부분은 서열 1-4로 나눌 수 있음
							 트립토판 있는 경우 : 서열 2번을 리보솜이 번역하고 있을 때 3,4번 서열이 루프 구조를 형성함 -> 그 뒤로 UUUUUU가 나와서 Rho 비의존적 종결이 이루어짐
							 트립토판 없는 경우 : 서열 1번에 트립토판을 지정하는 코돈이 여러개 나와서 리보솜의 번역이 멈춤 -> 2, 3번 서열이 루프를 형성함 -> 4번에는 UUUUUU가 없기 때문에 종결없이 계속 전사가 이루어짐
	진핵생물 유전자 발현의 조절
		염색체 상의 변화
			히스톤 수준 : 히스톤 H1의 결합 여부, 히스톤의 리신기에 아세틸화(리신의 양전하성 제거 -> 느슨해짐)
			DNA 수준 : CpG 서열의 시토신을 메틸화(단백질이 해당 서열 인식 못함) -> Heterochromatin이 됨
		전사 과정의 조절 : 전사인자가 조절 서열에 결합을 해야됨
			특수 전사인자 : 특정 유전자 조절 서열에 붙음 (아무 유전자를 다 조절하는게 아님) 생물에서 특정 유전자를 발현해야겠다 싶을 때 해당 특수 전사인자를 활성화시킴
				특수 전사인자가 붙는 서열 종류
					원거리 서열 : 증폭자 Enhancer(activator), 침묵자 Silencer(Repressor)
						Enhancer에 특수 전사인자가 붙으면 HMG라는 단백질이 DNA를 구부려서 Enhancer랑 TATA 박스를 가깝게 만듬 -> 히스톤아세틸화 효소(Euchromatin으로 만듦), 히스톤 재구조화 복합체(히스톤을 옆 쪽으로 치우는 효소)가 특수 전사인자에 붙음 -> 거기에 매개자(Mediator) 단백질이 붙음 -> 그러면 일반 전사인자가 TATA박스에 붙음
					근거리 서열 : 조절서열 Regulatory sequence (activator or Repressor)
			일반 전사인자 : TATA 박스에 결합하는 전사인자로 전사 개시할 때 RNA Polymerase랑 복합체를 이룸
		RNA 가공의 조절
			선택적 스플라이싱 : 한 개의 유전자가 상황에 따라 다양한 단백질을 만듦
		mRNA 수송 : mature mRNA의 3' UTR에 zip code라는 부분이 있는데 해당 부분을 참조해서 RNA가 세포 골격을 타고 옮겨짐
		mRNA의 수명 조절 : 단백질 양이 많이 필요하면 수명도 길게함. 3' UTR에 있는 정보에 따라 수명이 결정됨
		번역의 조절 : mRNA가 2차 구조(stem-loop, hairpin)를 형성하거나 단백질이 붙어서 리보솜이 mRNA에 붙어서 번역을 하는 과정에 영향을 줄 수 있음
		cf) 마이크로 RNA
			핵에서 RNA Pol II가 pri-miRNA(1차, primary miRNA)를 전사함 -> pri-miRNA에 루프가 생김 -> 드로샤라는 단백질이 와서 루프 부분만 잘라냄 -> 잘라진 루프 부분을 pre-miRNA라고 부름. pre-miRNA는 핵 밖으로 나감 -> 다이서(Dicer)가 pre-miRNA를 잘라서 짧은 dsRNA로 만듬 -> 그걸 RISC라는 단백질이 와서 그 중 한 가닥을 자신한테 붙이는데 그 복합체를 miRNA라고 부름 -> miRNA가 다른 RNA에 붙어서 완전 일치하면 그 부분을 잘라내버리고 적당히 일치하면 그냥 붙어서 리보솜이 번역 못하게 방해함
		cf) siRNA(small interering RNA) : Dicer, RISC 작용하는 건 똑같은데 외부에서 들어온 RNA가 Dicer에 의해서 짧은 dsRNA로 잘린 것을 siRNA라고 부름(RISC가 붙기 전)
		단백질 활성 조절 : 비가역, 가역, 조절 단백질, 알로스테리
		단백질 분해 : 수명이 다한 단백질에 유비퀴틴을 붙임 -> 유비퀴틴 붙은 단백질은 프로테아좀에서 짧은 올리고펩티드로 분해됨
돌연변이
	복제판 배양법(Replica plating) : 콜로니들이 형성된 배지를 벨벳 조각으로 찍어서 다른 배지에 옮기는 방법(도장찍는 느낌)
		복제판 배양법으로 배지를 옮기면서 야생형 세균을 기르고 있었는데 최소 배지에서 살지 못하는 영양 요구주가 생겨남 -> 자발적 돌연변이 (돌연변이는 원래 자연적으로 발생)
	돌연변이 종류
		염기 돌연변이 : 염기 치환, 염기 첨가, 염기 결실
			치환(Substitution)
				전위(Transition) : 퓨린이 퓨린으로 피리미딘이 피리미딘으로 바뀌는 것
				전좌(Transversion) :퓨린, 피리미딘 간에 바뀌는 것
			단백질의 영향에 따른 분류
				침묵 돌연변이(silent mutation) : 염기가 바뀌었는데 아미노산은 그대로인 경우
				미스센스 돌연변이(missense mutation) : 아미노산이 바뀐 경우
				넌센스 돌연변이(nonsense mutation) : 염기가 바뀌어서 종결 코돈이 된 경우
			아미노산의 영향에 따른 분류
				안바뀜 : 동의 돌연변이, 바뀜 : 비동의 돌연변이
			단백질 기능에 이상이 없는 돌연변이를 중립 돌연변이라고 부름
			첨가 또는 결실 돌연변이
				3의 배수가 아닌 수가 증감되면 frame shift mutation 일어날 수 있음
		염색체 구조 이상 돌연변이 : 결실, 역위, 중복, 전좌
			결실(deletion) : 염색체 일부 절편이 소실되는 현상
			중복(duplication) : DNA 재조합에 의해서 일부 절편이 추가, 반복되는 현상
			역위(Inversion) : 염색체의 유전 정보 배열이 뒤집히는 경우
			전좌(Translocation) : 한 염색체의 일부가 떨어져 나와 상동 염색체가 아닌 염색체에 삽입되는 경우
			상호 전좌 : 상동이 아닌 염색체 사이에 일부 절편을 교환하는 현상
		염색체 수 이상 돌연변이 : 염색체 수가 달라짐(염색체에서 정상적인 분리가 이루어지지 않은 경우)
			이수성 돌연변이 : 염색체가 제1 또는 2 감수분열에서 비분리되어서 핵상이 n-1 또는 n+1이 되는 현상
				상염색체 21 이상 : 다운증후군, XXY : 클라인펠터 증후군, XO : 터너 증후군
			배수성 돌연변이 : 모든 염색체가 제1 또는 2 감수분열에서 모두 비분리되어서 핵상이 2n이 되는 현상
	돌연변이 유발원
		DNA 복제의 실수 : DNA Polymerase에 있는 교정 기능(3'->5' 핵산말단가수분해 효소)이 가끔 놓칠 때도 있음
		유사 염기의 첨가 : DNA 복제할 때 유사 염기가 들어가서 나중에 한번 더 복제할 때 엉뚱한 염기쌍을 형성할 수 있음
		화학적 돌연변이 유발원 : 특정 화학물질이 염기에 화학변화를 일으켜서 잘못된 염기쌍을 형성할 수 있음
		삽입물질(intercalating agent) : 소수성이 강한 판상의 고리 구조 화합물이 DNA 사이에 끼어들어가서 DNA의 물리적 구조를 변형시키는 현상 -> 결실이나 첨가를 일으킬 수 있음 (frame shift)
			에티듐 브로마이드(EtBr; 자외선 쪼이면 형광 나타내서 아가로오스 겔 전기영동할 때 주로 사용함), 아크리딘 오렌지, 프로피디움 요오드화물
		물리적 돌연변이
			자외선 조사로 인한 피리미딘 이량체 : 피리미딘p피리미딘(TpT가 가장 잘 생김) 사이에 UV를 쪼이면 공유결합이 생길 수 있음 -> frame shift 생김
			방사선 조사 : 라디칼이 발생해서 DNA 내 공유결합이 파괴될 수 있음
			전이인자 : 전이인자가 jumping을 해서 유전자 중간에 끼어들어가면 돌연변이가 발생할 수 있음
	돌연변이 유발원 검사 - 에임스 테스트
		His^- 살모넬라균을 최소배지에 넣고 기르면 돌연변이에 의해 다시 히스티딘을 합성할 수 있게되는 적은 수의 돌연변이가 발생함 -> 이때 최소배지에 돌연변이 유발원 의심 물질을 같이 넣으면 더 많은 수의 돌연변이가 발생하게 돼서 콜로니 수가 더 늘어남
			일부 돌연변이원은 그 자체로 작용하지 않을 수도 있기 때문에 시토크롬 P450(간에서 해독하는 물질)을 넣어서 해독작용 후 돌연변이 기작을 하는지 확인해봐야함
	돌연변이 복구 기작 : DNA 수선 효소들이 망가진 DNA를 고침
유전자 재조합 기술
	클로닝 : 관심있는 DNA 부분을 vector에 넣어서 세포에 도입해서 발현하는 기술. 형질전환 생물에서 특정 유전자를 발현시켜서 유용한 물질을 대량생산할 수 있음
		기본 요소들
			제한 효소(restriction enzyme) : 특정 염기 서열을 인식해서 포스포다이에스테르 결합을 끊는 효소. 원래는 세포가 바이러스에 대항하기 위한 방어 기작에 사용됨 (자신의 DNA는 메틸화시켜서 잘 잘리게 함 -> 메틸화된 DNA를 가지는 바이러스는 제한 효소를 회피할 수 있음)
				핵산 내부 가수분해 효소로서 회문 구조의 특정 염기 서열을 인식해 절단함
				blunt end(ㅣㅣ느낌) : 끝 부분 상관없이 붙을 수 있음(비특이적)
				sticky end(ㄴ ㄱ 느낌) 형성 : 끝 부분 pair가 맞아야 붙을 수 있음(특이적)
				(ligase 처리해주면 붙일 수 있음)
				명명법 : 속명, 종소명, 주, 발견 순서 e.g.) EcoR I
			플라스미드
				ori : 복제원점
				선택 마커 e.g.) Amp^R(암피실린), Tet^R(테트라사이클린)
				MCS(Multi-cloning site) : 다양한 제한효소로 자를 수 있는 부분
				프로모터
			pBluescript : MCS가 LacZ(베타갈락토시데이스) 사이에 있음 -> LacZ^- 대장균을 X-gal이랑 암피실린있는 곳에 키움 -> X-gal이 분해되면 푸른색띔
		순서 : PCR로 특정 DNA 절편 증폭 -> Vector(일반적으로 플라스미드 사용)에 DNA 절편을 끼워 넣음(ligase로 붙임) -> vector로 대장균에 유전자 형질 전환시킴 (열 충격법(heat shock) : 칼슘 이온을 넣으면 음전하 띄는 세포벽이랑 DNA가 칼슘이온에 붙음 그때 가열하면 DNA가 세포 안으로 들어감, 전기 천공법) -> 대장균을 선별함
	형질전환 식물의 개발 : 아그로박테리움을 이용해서 유전자를 도입함
	형질전환 동물의 개발
	유전자 치료 : 바이러스 벡터 등을 이용해서 유전자를 발현시켜서 질병을 치료
	cf) 원형질체 융합 : 클로닝 아니고 두 종류의 식물 세포를 융합시키는 것 e.g.) 포마토
염기 서열 분석과 지놈 프로젝트
	생어의 방법
		서열을 알고 싶은 절편을 벡터에 클로닝하고 복제시킴-> 가열해서 denaturation시키고 방사선 표지된 프라이머를 제한 효소 쪽에 붙여줌 -> dNTP넣고 4개로 나누고 각각에 ddNTP를 넣어주고 DNA합성효소도 넣어줌 -> 아가로오스 겔 전기영동하고 표지된 것 확인
	좀 더 진보된 생어의 방법 : 나눈 DNA들에 ddNTP를 넣을 때 각각 다른 형광물질을 표지시켜서 넣음 -> 그 다음 그걸 한번에 전기영동시키고 끝에 도달한 형광을 인식해서 서열을 분석함
	라이브러리 개발 : DNA를 절편으로 잘라서 각각을 벡터에 넣은 다음 복제시키는 방법
		DNA 라이브러리에서 특정 절편을 찾는 방법 : 배지에 있는 라이브러리를 니트로셀룰로오스 막으로 옮김(복제판 배양법같은 느낌) -> 세포 파쇄 후 탐침을 넣음 -> 확인
			cf) 탐침(probe) : 특정 DNA 서열에 상보적인 핵산 조각으로 표지(형광 또는 방사선)를 시키면 위치를 확인할 수 있음
		cDNA 라이브러리 : 세포 파쇄를 해서 전체 mRNA를 얻은 후 poly dT 프라이머를 넣고 역전사효소로 cDNA를 만들어서 라이브러리를 개발
	지놈 프로젝트 : 어떤 생물의 지놈 DNA 라이브러리를 만듦
		염색체 보행법(chromosome walking) : 2개의 DNA를 서로 다른 위치에서 절편으로 자른 후 서로를 참조해서 절편 간의 순서를 찾는 방법
유전자와 발현의 분석
	써던 블랏팅 : 샘플 내 특정 DNA의 존재 여부, 양, 크기를 확인하는 방법
		1. 해당 세포의 지놈 DNA를 적당한 제한효소를 이용해 자름
		2. 아가로오스 겔 전기영동을 하고 겔을 알칼리 용액에 담궈서 ssDNA로 분리시킴
		3. 모세관 현상을 통해 DNA를 NC 필터로 옮김
		4. DNA 탐침(관심있는 DNA 서열)을 넣고 필름을 감광해서 탐침의 위치를 확인
		(탐침이 붙을 때 DNA 서열 1-2개 정도 다른 건 그냥 붙을 수 있음)
	용어 정리
		RFLP(restriction fragment length polymorphism) : 각 개체 마다 제한 효소를 통해 DNA를 잘라서 전기 영동을 했을 때 밴드 패턴이 다르게 나타나는 현상
		STR(short tandem repeat) : 염색체 내 특정 위치에서 개체 마다 다르게 반복되어 나타나는 짧은 염기 서열
	노던 블랏팅 : 샘플 내 특정 mRNA의 존재 여부, 양, 크기를 확인하는 방법
		써던 블랏팅이랑 거의 비슷함. DNA가 mRNA로 바뀐거고 이미 외가닥이라 알칼리 용액 담굴 필요없고 탐침을 DNA 탐침(RNA가 분해되기 쉬워서 그럼. 자연에 DNase는 얼마 없는데 RNase는 여기저기 널림)을 넣어주면 됨
	RT-PCR : mRNA를 cDNA로 바꿔서 샘플 내 특정 mRNA의 존재 여부, 양, 크기를 확인하는 방법
		1. 개체에서 전체 mRNA를 추출
		2. 역전사 효소, poly dT 프라이머를 넣고 cDNA로 바꿈
		3. 관심있는 서열의 양 끝으로 프라이머를 짠 다음 PCR 시킴
		4. 아가로오스 겔 전기 영동시킨 다음 밴드 진하기로 양 확인
	DNA 마이크로어레이(Microarray, DNA chip assay) : 세포 마다 전사된 여러 mRNA들의 발현 패턴을 한번에 분석하는 방법
		1. 칩의 구멍에 탐침을 심음(구멍마다 한 종류, 한 구멍에는 똑같은 걸 여러개 심음)
		2. 각 세포에서 전체 mRNA를 얻음
		3. mRNA를 역전사 시켜서 cDNA를 얻는데 세포 마다 각각 다르게 표지된  dNTP를 넣음
		4. RNase로 mRNA는 다 분해해서 ssDNA를 얻음
		5. ssDNA를 chip에 뿌리고 어디에 hybridization이 되는지 확인함 -> 색상, 진하기에 따라 어떤 세포에서 유전자가 발현되었는지와 그 발현량을 비교할 수 있음
